My git prompt
=============

Yet another git prompt with emphasis on BASH (which is my default shell).

A project to learn a bit more about C.

Dependencies:
-------------
 * C compiler
 * gnu make
 * meson buildsysem

Start hacking:
--------------
Git clone this repository and run:
```
make init
```

To clean all build files:
```
make clean
```

To clean all meson setup files:
```
make clean-all
```
